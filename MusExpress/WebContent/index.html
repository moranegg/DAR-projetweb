Recherchez vos musées favoris en Ile de France

Consultez les informations des musées et les affluences en temps réel afin d'éviter de longues files d'attente.

MusExpress est un site communautaire !

Vous comptez aller à une exposition dans un musée ? Partagez votre temps d'attente afin d'en faire profiter les autres utilisateurs.

Vous ne savez pas quel musée visiter ? Nos recommandations sont faites pour vous.

MusExpress vous propose de visiter des musées où vous n'aurez pas à attendre longtemps. Nous prenons également compte de la météo actuelle pour vous éviter d'attendre sous la pluie :)

The source code for this application is available in this repository on GitHub.
